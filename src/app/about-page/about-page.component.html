About Warmama-Stats

Content

  1. Game Page

    This page displays a game. It includes the end-screen scoreboard, player stats, and additional information, such as the player with the most frags, score, damage, etc.

    You can hover over a player in the scoreboard to view their stats. You can also click on a player to go to their player page.

  2. Game Browser

    This page lets you browse the games in the database. You can filter by gametype, map, and player name. Clicking on a game in the table will take you to the game page.

  3. Player Page

    This page shows you statistics for a certain player. Individual players are identified by their nickname (this is the only way).

    You can view graphs of their weapon accuracy and damage across all games, filterable by gametype.

  4. Player Browser

    This page lets you browse through every player in the database. You can filter the list by name. Clicking on a player will show their basic stats, as well as 10 of their most recent games.

    You can click on a recent game to view that game page. There is also a link to the player page.

Development

This project is a work-in-progress and is developed in my spare time.

The frontend is built using Angular.

The backend uses node/express for the api endpoints, and pulls data from a MySQL database.

The project is open-source and available on GitHub; If you would like a link, just ask me. This is my first major project in the world of web development, so be warned :)
